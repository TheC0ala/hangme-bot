# HangMeBot
>Telegram Bot "Hangman Game"

## Set up

_Install_ all dependencies for the telegram bot:

```bash
pip install pyTelegramBotAPI
```

**Important!** Make sure you fill the [config file](config.py)!

## Run it

```bash
python bot.py
```
